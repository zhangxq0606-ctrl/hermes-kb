💡 关联底层认知思考：[[insight_Superpowers介绍]]

🔧 【用途定位】
- 解决 AI 编程工具“问一句答一句”的低效问题，适用于需要独立完成项目的场景（如网站开发、复杂功能实现）。

📋 【核心方法】
1. 安装 Claude Code 后，运行命令：`/plugin install superpowers@claude-plugins-official`，自动生效。
2. 其他 AI 编程工具（Cursor、Codex 等）也基本支持类似插件安装方式。
3. 使用流程：先让 Superpowers 提问澄清需求 → 生成详细 plan → 按 plan 分步执行。

⚠️ 【避坑指南】
- 确保 AI 编程工具支持插件系统，否则 Superpowers 无法生效。
- 安装命令需在对应工具的终端或插件面板中执行，而非系统终端。
