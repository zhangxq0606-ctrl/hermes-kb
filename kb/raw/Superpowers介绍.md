今天给大家推荐一个超牛逼的开源项目 superpowers。

项目地址：https://github.com/obra/superpowers

现在大部分人用 AI 编程工具帮你做网站或者项目之类的，大多数人还停留在「问一句、答一句」的阶段。

你让它写个函数，它写完了。你让它改个 bug，它改了。看着挺智能，但总觉得差点意思，而且每次问答都会消耗 token，它像个听话的实习生，而不是一个能独立完成项目的资深工程师。

而 superpowers 就是一套给 AI 编程助手装的「资深工程师」。

比如你现在要做一个网站，那么 superpowers 不会上来就写，会一步步问你一堆问题，确保你先想清楚要做什么。然后第二步，他会再出一份详细的 plan 计划，最后再按照 plan 一步步动手执行。

可以说，他就是一个完整的高级工程师的工作方式。

我今天用 claude code 搭载 superpowers 让他帮我做一个简单的小网站测试，实际体验下来就两个字：丝滑。

你们体验过之后，就能知道有多牛逼了。

claude code 下载很简单，一行代码：

/plugin install superpowers@claude-plugins-official

之后自动生效。

如果你用的其他 AI 编程工具，Cursor/Codex 等也基本都支持。

赶紧去体验吧，之后你一定会回来感谢我。
